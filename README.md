demo1:

demo2:
做一个登录注册的 DEMO 界面
1.用户界面设计
登录页面:包含用户名/密码输入框，以及登录、记住我和注册按钮。
注册页面:包含用户名/密码/确认密码。
响应式设计:确保页面在不同设备(如桌面、平板、手机)上都能良好显示。

数据验证。前端验证:实时检查输入是否符合格式要求，如邮箱格式、密3.
码强度等。
